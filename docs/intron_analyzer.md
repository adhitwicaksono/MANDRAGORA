# Intron Analyzer

`mandragora intron` infers introns from exon coordinates in a genome annotation file and generates intron-level and gene/transcript-level statistics.

This module is designed for exploring gene structure complexity in unusual plant genomes, especially large, repeat-rich, and parasitic plant genomes.

---

## Purpose

The intron analyzer helps answer questions such as:

| Question | Output |
|---|---|
| How many introns are present in the annotation? | `intron_stats.tsv` |
| Which genes/transcripts contain introns? | `gene_intron_summary.tsv` |
| Which genes have very long introns? | `gene_intron_summary.tsv` |
| What are the coordinates of inferred introns? | `introns.bed` |
| Are there single-exon genes? | `gene_intron_summary.tsv` |

This is especially useful for genomes where long introns may reflect:

| Possibility | Interpretation |
|---|---|
| Real biology | Large introns, repeat-rich introns, unusual gene architecture |
| Annotation artifact | Incorrect exon joining, fragmented assembly, poor gene prediction |
| Repeat-driven inflation | Introns expanded by transposable elements or repetitive regions |
| Assembly issue | Exons joined across questionable scaffold regions |

MANDRAGORA does not automatically decide which explanation is correct. It provides a diagnostic report for manual interpretation.

---

## Basic Usage

```bash
python -m mandragora.cli intron \
  --annotation examples/toy_annotation.gff3 \
  --outdir results/intron
```

Arguments:

| Argument | Description |
|---|---|
| `--annotation` / `-a` | Input genome annotation in GFF3/GTF-like format |
| `--outdir` / `-o` | Output directory |

---

## Input

The current recommended input is GFF3.

Required feature hierarchy:

```text
gene
mRNA or transcript
exon
```

Example:

```gff3
##gff-version 3
scaffold_1	toy	gene	101	500	.	+	.	ID=gene1
scaffold_1	toy	mRNA	101	500	.	+	.	ID=gene1.t1;Parent=gene1
scaffold_1	toy	exon	101	150	.	+	.	ID=gene1.exon1;Parent=gene1.t1
scaffold_1	toy	exon	251	300	.	+	.	ID=gene1.exon2;Parent=gene1.t1
scaffold_1	toy	exon	401	500	.	+	.	ID=gene1.exon3;Parent=gene1.t1
```

MANDRAGORA infers introns as the regions between consecutive exons.

## Output Files

Running the intron analyzer produces:

```text
results/intron/
├── introns.bed
├── gene_intron_summary.tsv
└── intron_stats.tsv
```

---

## `introns.bed`

This file contains inferred intron coordinates.

Example:

```bed
scaffold_1	150	250	gene1.intron1	0	+
scaffold_1	300	400	gene1.intron2	0	+
scaffold_1	650	800	gene2.intron1	0	-
```

Columns:

| Column | Meaning |
|---|---|
| `chrom` | Scaffold/chromosome |
| `start` | Intron start, 0-based |
| `end` | Intron end, half-open |
| `name` | Intron ID generated by MANDRAGORA |
| `score` | Placeholder score, currently `0` |
| `strand` | Strand inherited from transcript/gene |

---

## `gene_intron_summary.tsv`

This file summarizes introns per transcript.

Example:

```tsv
gene_id	transcript_id	chrom	strand	exon_count	intron_count	total_intron_bp	max_intron_length
gene1	gene1.t1	scaffold_1	+	3	2	200	100
gene2	gene2.t1	scaffold_1	-	2	1	150	150
gene3	gene3.t1	scaffold_2	+	1	0	0	0
```

Columns:

| Column | Meaning |
|---|---|
| `gene_id` | Gene ID |
| `transcript_id` | Transcript or mRNA ID |
| `chrom` | Scaffold/chromosome |
| `strand` | Gene/transcript strand |
| `exon_count` | Number of exons in the transcript |
| `intron_count` | Number of inferred introns |
| `total_intron_bp` | Total intron length for that transcript |
| `max_intron_length` | Longest intron in that transcript |

---

## `intron_stats.tsv`

This file summarizes all inferred introns.

Example:

```tsv
metric	value
total_transcripts	3
total_introns	3
total_intron_bp	350
min_intron_length	100
max_intron_length	150
mean_intron_length	116.666667
median_intron_length	100
```

Metrics:

| Metric | Meaning |
|---|---|
| `total_transcripts` | Number of transcript models analyzed |
| `total_introns` | Number of inferred introns |
| `total_intron_bp` | Total bp covered by introns |
| `min_intron_length` | Shortest inferred intron |
| `max_intron_length` | Longest inferred intron |
| `mean_intron_length` | Mean intron length |
| `median_intron_length` | Median intron length |

## Coordinate Logic

GFF3 input:

```text
exon 101–150
exon 251–300
```

Internal BED-style representation:

```text
exon 100–150
exon 250–300
```

Inferred intron:

```text
150–250
```

This corresponds to GFF3 positions:

```text
151–250
```

## Interpretation for Weird Plant Genomes

Very long introns may indicate interesting biology, but they may also indicate annotation problems.

Useful follow-up questions:

| Observation | Possible next check |
|---|---|
| Gene has extremely long intron | Check repeat content inside intron |
| Many single-exon genes | Check gene prediction quality |
| Gene span is huge but CDS is short | Check for annotation inflation |
| Introns concentrated on specific scaffolds | Check assembly/scaffold quality |
| Introns overlap many repeats | Run `mandragora repeat-overlap` or future repeat-intron analysis |
